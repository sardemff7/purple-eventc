purple-eventc

An eventd client plugin

purple-eventc provides events dispatching for purple-events to eventd.

Dependencies

  • Pidgin (2.6.0 or later)
  • purple-events
  • libeventc (distributed with eventd)

Git repository

The repository is hosted on GitHub: purple-eventc Git repository

You can clone the project with Git by running:
$ git clone git://github.com/sardemff7/purple-eventc
